##  设计特点

### 色彩系统
- **主色调**: 紫色渐变 (#667eea → #764ba2) - 营造神秘感和专业感
- **四色系**: 
  - NT理性者: 深蓝色/紫色 (#6366f1)
  - NF理想主义者: 绿色/青色 (#10b981)
  - SJ护卫者: 橙色/金色 (#f59e0b)
  - SP艺术创作者: 红色/粉色 (#ef4444)

### 情绪能量条
- **安全区** (60-100): 绿色渐变 (#10b981 → #a8e063)
- **警告区** (30-59): 黄色渐变 (#f59e0b → #fbbf24)
- **危险区** (0-29): 红色渐变 (#ef4444 → #f87171)

### 交互设计
- 卡片悬浮效果 (transform: translateY(-3px))
- 按钮点击反馈 (box-shadow)
- 进度条平滑过渡 (transition: width 0.5s ease)
- 关键动画 (弹跳、脉冲、浮动)

### 响应式适配
- 桌面端: 3列网格布局
- 平板/移动端: 自动切换为单列
- 手机模拟器: 375×667标准尺寸
